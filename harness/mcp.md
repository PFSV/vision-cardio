# MCP Usage

Use MCP-backed tools when they reduce uncertainty or eliminate stale recall.

## GitHub

Use GitHub tools for:

- repository inspection,
- PR and issue review,
- patch inspection,
- commit and branch history,
- code search when the repo is available.

## Hugging Face

Use Hugging Face tools for:

- model search,
- dataset discovery,
- documentation lookup for HF libraries and hosted assets.

## Web

Use web search only when:

- the question is time-sensitive,
- the source must be official,
- the repo does not contain the needed artifact.

## Rule

Do not guess at external facts when a connector can verify them.
